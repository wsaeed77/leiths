=== WooCommerce Opayo Form, Opayo Direct and Opayo Pi Gateway ===

Extends WooCommerce. Provides a Opayo Form, Opayo Direct and Opayo Pi gateway for WooCommerce. http://www.opayo.com. Email support@chromeorange.co.uk with any questions. If you need technical support please turn on debugging and supply the contents of the debug emails.

Please copy the 'woocommerce-gateway-sagepay-form' to your wp-content/plugins folder to install.

Protocol 3 requires PHP mcrypt be installed on your server
