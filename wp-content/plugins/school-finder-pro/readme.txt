A plugin to manage and search through a database of schools. Upload CSV files with school data and provide a searchable dropdown on the frontend.

== Description ==

School Finder Pro allows administrators to upload and manage a database of schools via CSV files, and provides a searchable dropdown interface on the frontend for users to find and select schools.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/school-finder-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> School Finder Pro screen to configure the plugin

== Frequently Asked Questions ==

= How do I upload school data? =

Go to School Finder Pro in the admin menu and upload your CSV file.

= How do I display the search on my site? =

Use the shortcode [school_finder] on any page or post.
